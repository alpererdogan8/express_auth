## A simple authentication application for Express

### Install 
- - -
 `$ npm install `
### Usage 
- - -
`$ npm start ` or `$ npm run watch `

### Endpoints

`URL = http://localhost:9000/api/v1/auth/`

#### Guest
- - -
```
  - /
  - /name="name"
  - /info
```

#### Auth
- - -
  ``` 
   /
   /register
   /login
   /user
   /private
   /private/"user"
   /logout
   ```
